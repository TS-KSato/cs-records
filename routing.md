# 主導区分と対応の前提

各記録ファイルでは tags の「主導」に TS または CY を書く。
TS / CY は主導会社の区分を表す内部指標。

## 主導区分
- TS / CY の2区分がある。
- システムは TS・CY で共通（照合手順・画面の挙動は同じ）。
- 違うのは広告集客と、電話の受電・折り返しの担当体制。

## 対応経路の前提
- メール: 決済代行系（GMO / SB Payment）は自社対応、キャリア決済系は外部が一次対応。
- 電話: 一次受電と折り返しは外部が担う。具体的な担当の割り当ては、
  公開しない運用情報として別途（このファイルには記載しない）。

## 補足
- 一次受電の一部は受けるのみで、ユーザー情報を確認できない区分がある。
